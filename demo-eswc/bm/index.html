Simply drag the following link to your browser bookmark bar.

Find something to annotate

In this demo we encourage you to annotate online news (blogs, newspapers, etc.). Your annotations will be used to drive the faceted search.

You could start from one of the suggested web sites:

  • http://www.technewsworld.com/
  • http://www.sciencedaily.com/
  • http://www.bbc.com/news/
  • http://www.theguardian.com/
  • http://europeandaily.com/
  • http://www.neurope.eu/
  • http://www.euronews.com/
  • http://www.aljazeera.com/
  • http://www.ansa.it/english/news/
  • http://www.corriere.it/english/
  • http://www.voanews.com/
  • http://techcrunch.com/

    • Load Pundit and annotate

    Once you find something interesting and you are ready to annotate click the "Pundit ESWC demo" button in your browser bookmark bar.

    Warning: While we the bookmarklet allows you to annotate virtually any web page, it is still under development and it might have technical problems loading on "complex" web pages. You could notice wrong styles or problems in the UI.

    Tip: If the bookmarklet is not loding properly, try to refresh the page in your browser and wait for the all page to be loaded, then give it a second try.

    Tip: If the automatic annotation suggestions does not work when you select a big text, please try to select single paragraphs at a time.
