Data crowdsourcing with Pundit.

Welcome to the Pundit demo at ESWC 2015.


The idea in a sentence

"We have web documents and a community of users, and we want them to contribute in building a knowledge graph that can be explored (e.g. via a faceted browser)."

Pundit who?

Pundit is a semantic web annotation system that allows you to create structured annotations on top of web documents. Annotations are RDF triples that can be esily merged to form in fact a knowledge graph.

Pundit can be configured to address specific communities both in terms of UI and annotation vocabularies. Once properly configured it can be deployed as a Javascript library, to be added to your site to enable annotation of your web pages, or as a bookmarklet, to be distributed to your users to annotate any (almost) web page.

Try the demos!

To demonstrate data crowsouring with Pundit we set up two demos:

1. Open scenario demo

Use the Pundit bookmarklet to annotate online news from a variety of web sites and build a collaborative faceted search. - Try it

2. Controlled scenario demo

Annotate the documents collected in a toy digital library and use the built-in Pundit tool to enrich the faceted navigation. - Try it
